# 💎 Mahalaxmi Jewellers Website

A full-stack dummy project for a jewellery website, built to practice and demonstrate front-end and back-end development skills.

---

## 📌 About the Project

This is a **full-stack web project** for a jewellery brand **Mahalaxmi Jewellers**.  
The project is created for learning and portfolio purposes, covering both UI design and backend functionality.

It focuses on:
- Clean and modern UI
- Structured front-end development
- Backend integration (planned / in progress)
- Real-world project structure

---

## 🚀 Tech Stack

### Frontend
- **HTML5** – Structure  
- **CSS3** – Styling (Flexbox & Grid)  
- **JavaScript (ES6)** – Interactivity  

### Design
- **Figma** – UI/UX design reference
